HTML (HyperText Markup Language) є мовою розмітки, що використовується для створення веб-сторінок. Вона дозволяє визначати структуру та зовнішній вигляд контенту на веб-сторінці, включаючи текст, зображення, відео, посилання та інші елементи. Розглянемо основні компоненти HTML більш детально:

1. Теги (Tags): Основна будівельна одиниця HTML. Вони визначають різні елементи на сторінці. Наприклад, для параграфу, h1 - h6 для заголовків різного рівня, для зображень.

2. Елементи (Elements): Кожен HTML-тег утворює елемент. Елемент може містити текстовий контент або інші вкладені елементи.

3. Атрибути (Attributes): Деякі теги можуть містити атрибути, які надають додаткову інформацію про елемент. Наприклад, атрибут src в тезі вказує шлях до зображення, яке потрібно відобразити.

4. Структура сторінки: HTML-документ зазвичай складається з декількох основних елементів

!DOCTYPE html: Оголошення типу документа, що вказує на версію HTML.

html: Кореневий елемент, що включає весь вміст сторінки.

head: Розділ, що містить метадані та інші технічні деталі про сторінку, такі як заголовок (title), підключення стилів (link) та скриптів (script).

body: Основний вміст сторінки, включаючи текст, зображення, таблиці, форми тощо.
